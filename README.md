## Version
- python 3.9
- 참고 사이트: [https://docs.python.org/](https://docs.python.org/)


## Install
##### virtualenv
```
python -m venv ./venv

# window
cd venv\Scripts
activate

# linux
cd venv/bin
source activate

(venv) $ pip install -r requirements.txt
```
##### basic
```
pip install -r requirements.txt
```


## Run Server - [http://localhost:5000](http://localhost:5000)
##### virtualenv
```
# window
cd venv\Scripts
activate

# linux
cd venv/bin
source activate

(venv) $ python main.py
```

##### basic
```
python main.py
```
